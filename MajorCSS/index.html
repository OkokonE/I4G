Difference between Padding, Margin and Border?:

Padding is used to create space around an element's content, inside of any defined borders. Margin properties are used to create space around elements, outside of any defined borders. Border property is a shorthand syntax in CSS that accepts multiple values for drawing a line around the element it is applied to.
